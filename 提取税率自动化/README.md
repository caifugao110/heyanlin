# 提取税率自动化工具

## 1. 使用说明

### 1.1 安装 VBA 支持
关闭 WPS，解压压缩包 `vba\wps2019vba.rar`，然后安装里面的 `wps2019vba.exe` 文件。

### 1.2 加载宏文件
1. 打开 WPS，点击「工具」→「加载项」；
2. 在选项框中点击「浏览」；
3. 选择 `加载宏文件xlam` 目录下的「提取税率自动化YYMMDD.xlam」；
4. 加载完成后，在加载项选项框中将该文件勾选。

### 1.3 执行提取税率
1. 打开包含「销售台账」工作表的表格文件；
2. 按快捷键 `Ctrl+Shift+W` 即可自动完成税率提取。

---

## 2. 加载宏文件制作方法

1. 打开 `过程文件.xlsm`，点击「工具」→「VB编辑器」；
2. 将 `ExtractTaxRate.bas` 中的代码复制到 `过程文件.xlsm` 的模块中；
3. 点击「运行宏」→「选项」，设置快捷键为 `Ctrl+Shift+W`；
4. 将文件另存为「Excel 加载宏(*.xlam)」格式；
 5. 保存到 `加载宏文件xlam` 目录下，命名为「提取税率自动化YYMMDD.xlam」（YYMMDD 为日期，如 260615）。

## 功能说明

该工具实现以下功能：
- 自动在「销售台账」工作表中查找表头行（在第1-100行范围内搜索）
- 执行前验证以下必要列是否存在：
  - 合同号
  - 税率
- 提取合同号-税率信息，按「合同号 + 税率」组合去重
  - 税率相同判定规则：税率值四舍五入保留5位小数后相等即为相同
  - 合同号相同但税率不同，会分别记录
- 税率列中的非数值内容（如"免税"、"零税率"等）所在行自动跳过
- 结果输出到同一工作簿的「税率一览表」工作表中，已存在则覆盖
- 自动设置列宽、对齐、边框、筛选，并加粗表头
- 完成后自动跳转到「税率一览表」工作表，并显示耗时
