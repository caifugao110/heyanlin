# 删除重复项自动化工具

## 1. 使用说明

### 1.1 安装 VBA 支持
关闭 WPS，解压压缩包 `vba\wps2019vba.rar`，然后安装里面的 `wps2019vba.exe` 文件。

### 1.2 加载宏文件
1. 打开 WPS，点击「工具」→「加载项」；
2. 在选项框中点击「浏览」；
3. 选择 `加载宏文件xlam` 目录下的「删除重复项自动化YYMMDD.xlam」；
4. 加载完成后，在加载项选项框中将该文件勾选。

### 1.3 执行删除重复项
1. 打开需要处理的表格文件；
2. 按快捷键 `Ctrl+Shift+Q` 即可自动完成重复项删除。

---

## 2. 加载宏文件制作方法

1. 打开 `过程文件.xlsm`，点击「工具」→「VB编辑器」；
2. 将 `DeleteDuplicateRows.bas` 中的代码复制到 `过程文件.xlsm` 的模块中；
3. 点击「运行宏」→「选项」，设置快捷键为 `Ctrl+Shift+Q`；
4. 将文件另存为「Excel 加载宏(*.xlam)」格式；
5. 保存到 `加载宏文件xlam` 目录下，命名为「删除重复项自动化YYMMDD.xlam」（YYMMDD 为日期，如 250604）。

## 功能说明

该工具实现以下功能：
- 根据全部5列的组合判断重复数据（预算合同编号、合同预算项目、累计已用预算、调整后预算金额.、最终预算（累计已用与调整后预算取大值））
- 自动查找表头行（在第1-100行范围内搜索）
- 执行前验证以下必要列是否存在：
  - 预算合同编号
  - 合同预算项目
  - 累计已用预算
  - 调整后预算金额.
  - 最终预算（累计已用与调整后预算取大值）
- 自动删除重复行
- 显示删除统计和程序耗时信息
